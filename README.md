# unity2godot
Convert Unity projects to Godot 

Dependencies:
- [glfw 3.4](https://github.com/glfw/glfw/releases/tag/3.4)
- [imgui v1.92.6](https://github.com/ocornut/imgui/releases/tag/v1.92.6)
- [miniz 3.1.1](https://github.com/richgel999/miniz/releases/tag/3.1.1)
- [Native File Dialog Extended v1.3.0](https://github.com/btzy/nativefiledialog-extended/releases/tag/v1.3.0)
- [ufbx v0.20.0](https://github.com/ufbx/ufbx/releases/tag/v0.20.0)

Godot 4.6.1 is using:
- ufbx v0.20.0
